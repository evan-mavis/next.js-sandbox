## Recipe Micro App

A simple recipe search application that allows users to search and browse recipes using real data from the Spoonacular API.

**Completion Time**: 1 hour and 58 minutes

### Goals

1. **Pixel-perfect implementation**: Match the v0 design mockup with high fidelity
2. **Next.js SearchParams**: Implement dynamic URL-based search using Next.js `searchParams` API
3. **Server Components**: Fetch recipe data from Spoonacular API using Next.js Server Components
